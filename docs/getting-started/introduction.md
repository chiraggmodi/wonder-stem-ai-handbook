# Introduction

Welcome to the **Wonder STEM AI Handbook**.

This handbook documents the complete production workflow used to create high-quality STEM educational content with AI.

Unlike traditional prompt collections, this project provides an entire production factory composed of specialized AI agents that collaborate to research, verify, script, audit, and prepare educational content for publishing.

The goal is simple:

> Produce educational content that is accurate, engaging, scalable, and reproducible.

---

## Why this project exists

Creating educational videos involves many repetitive tasks:

- Topic research
- Fact verification
- Script writing
- Visual planning
- Thumbnail creation
- Metadata generation
- Publishing

Doing everything manually quickly becomes slow and inconsistent.

Wonder STEM Factory automates the entire pipeline while keeping humans in control of every important decision.

---

## Core Principles

The project follows several principles.

### Accuracy First

Every scientific claim must be verified before publication.

---

### Repeatable Workflows

Every AI agent performs a single responsibility.

Instead of one enormous prompt, each stage performs one job extremely well.

---

### Structured Outputs

Every workflow produces structured artifacts.

Examples:

- Research Brief
- Production Manifest
- Script
- Verification Report
- Thumbnail Plan

These outputs become inputs for the next stage.

---

### Human Approval

AI assists.

Humans approve.

Every important artifact should be reviewed before publishing.

## Project Philosophy

Think of this repository as a software project.

Instead of compiling code...

...it compiles educational content.

Each workflow is a reusable module.

Each artifact is version-controlled.

Each AI agent behaves like a microservice with a clearly defined responsibility.

Together they create a scalable educational production system.
