# Installation

This repository contains documentation only.

No software installation is required to read the handbook.

---

## Clone Repository

```bash
git clone https://github.com/chiraggmodi/wonder-stem-ai-handbook.git

cd wonder-stem-ai-handbook
```

---

## Install MkDocs

```bash
pip install mkdocs-material
```

Or

```bash
pip install -r requirements.txt
```

---

## Serve Documentation

Run the local documentation server.

```bash
mkdocs serve
```

Open:

```
http://127.0.0.1:8000
```

---

## Build Static Site

```bash
mkdocs build
```

Generated site:

```
site/
```

---

## Deploy

GitHub Pages

```bash
mkdocs gh-deploy
```

or

GitHub Actions.
