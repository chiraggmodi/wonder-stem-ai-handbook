# Topic Assessment

## Purpose

Evaluate whether a STEM topic is worth researching by assessing its educational value, audience appeal, and viral potential.

---

## Position in Workflow

```
User Input
    ↓
Topic Assessment
    ↓
Research Brief
```

---

## Input

| Variable | Description |
|----------|-------------|
| User Input | Topic entered by the user |

---

## Output

Produces a structured report containing:

- Core Idea
- Audience Analysis
- Viral Potential
- Weakness Analysis
- Content Improvements
- Platform Fit
- Audience Prediction
- Demand Validation
- Wonder STEM Optimization

---

## Responsibilities

- Evaluate viral potential
- Identify audience
- Improve topic framing
- Recommend stronger hooks
- Optimize for Wonder STEM

---

## Non-Responsibilities

This GEM does NOT:

- Research scientific facts
- Write scripts
- Generate production plans
- Verify claims

Those are handled by later GEMs.
