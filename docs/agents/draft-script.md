# Draft Script

> Version: 1.0.0

---

# Purpose

The Draft Script GEM transforms an approved Blueprint into a complete, narration-ready STEM video script.

Its sole responsibility is to convert the educational blueprint into engaging spoken narration while preserving scientific accuracy, audience retention, and storytelling quality.

It does not perform research, verify facts, or redesign the story architecture.

---

# Position in Workflow

```text
Blueprint
      ↓
Draft Script
      ↓
Verification Report
```

---

# Input

| Variable | Description |
|----------|-------------|
| Blueprint Handoff | Complete storytelling blueprint generated by the Blueprint GEM |

---

# Output

Produces a narration-ready Markdown script containing:

- Hook
- Central Question
- Investigation
- First Discovery
- Deeper Mystery
- Major Reveal
- Bigger Implication
- Final Revelation

---

# Responsibilities

This GEM is responsible for:

- Writing narration
- Managing curiosity loops
- Maintaining audience retention
- Following the Blueprint
- Preserving scientific accuracy
- Matching the requested runtime
- Producing a voice-over ready script

---

# Non-Responsibilities

This GEM does **not**:

- Research topics
- Add new scientific facts
- Verify claims
- Design the story
- Select the audience
- Generate production instructions
- Create SEO assets

Those tasks belong to other GEMs.

# Design Principles

- Blueprint is the single source of truth
- Narration only
- Investigation before explanation
- Curiosity before answers
- Scientific integrity over drama
- Educational storytelling
- Spoken language only

---

# Key Writing Rules

The generated script must:

- Follow the Blueprint exactly
- Match the requested runtime
- Resolve all curiosity loops
- Preserve scientific guardrails
- Avoid lecture-style narration
- Feel like an investigation
- End with a meaningful perspective shift

---

# Known Limitations

- Script quality depends entirely on the Blueprint.
- Missing Blueprint sections prevent script generation.
- Human review is recommended before production.
