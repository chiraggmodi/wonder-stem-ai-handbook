# Factory Workflow

The production workflow is designed as a linear pipeline with optional review loops.

## Primary Workflow

1. Topic Assessment
2. Research Brief
3. Blueprint Handoff
4. Draft Script
5. Verification Report
6. Audit Report
7. Revision Manager
8. Thumbnail & Title Optimizer
9. Production Planner
10. Scene Breakdown
11. Google Flow Prompt Generator
12. Publishing Package

## Review Gates

Human review is recommended after:

- Research
- Script
- Verification
- Final production package

## Benefits

- Clear ownership of each stage
- Easier quality control
- Repeatable results
- Faster onboarding for collaborators
