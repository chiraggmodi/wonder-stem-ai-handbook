# Factory Artifacts

Artifacts are the structured documents produced at each stage of the Wonder STEM Factory.

## Common Artifacts

| Workflow | Output Artifact |
|----------|-----------------|
| Topic Assessment | Topic Assessment Report |
| Research | Research Brief |
| Planning | Blueprint Handoff |
| Writing | Draft Script |
| Verification | Verification Report |
| Audit | Audit Report |
| Revision | Updated Script |
| Thumbnail | Thumbnail Brief |
| Production | Production Manifest |
| Publishing | Upload Package |

## Why Artifacts Matter

Artifacts provide:

- Traceability
- Version history
- Easier collaboration
- Review checkpoints
- Reusable inputs

## Storage

Each artifact should be stored in a predictable location and, when possible, version-controlled alongside the documentation or production assets.

## Best Practices

- Use consistent templates
- Preserve previous versions
- Reference source material
- Record approval status
- Keep artifacts focused on a single purpose
