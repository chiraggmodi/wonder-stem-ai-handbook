# Wonder STEM Factory

The Wonder STEM Factory is the central production system behind the handbook.

Rather than relying on a single AI prompt, the factory divides content creation into specialized stages performed by dedicated AI agents.

Each stage has a clear responsibility, producing structured outputs that feed into the next stage.

## Factory Goals

- Create consistent educational content
- Improve scientific accuracy
- Reduce repetitive work
- Scale production efficiently
- Keep humans in control

## Core Components

- Workflow architecture
- AI agents
- Artifacts
- Approval checkpoints
- Publishing pipeline
